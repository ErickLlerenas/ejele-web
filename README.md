# Ejele Web

Landing page para Ejele - Software gratis para restaurantes.

## Características

- Landing page enfocada en descargar la aplicación
- Software 100% gratis
- Diseño moderno con Tailwind CSS
- React + TypeScript + Vite

## Instalación

```bash
npm install
# o
yarn install
```

## Desarrollo

```bash
npm run dev
# o
yarn dev
```

## Build

```bash
npm run build
# o
yarn build
```
